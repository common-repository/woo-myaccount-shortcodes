== Description ==

WC Myaccount Shortcodes is an addon that provides you the shortcodes for woocommerce myaccount page e.g. current logged in user downloads, orders, account details and address etc.
These shortcodes are very useful if you want to show your users their woocommerce orders or downloads individually and seprately from their myaccount page.
